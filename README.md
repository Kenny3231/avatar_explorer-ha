# Avatar Explorer 🧔🤵👩‍❤️‍👨

Intégration personnalisée pour gérer des Bitmojis dynamiques sur Home Assistant.

## Installation
1. Copiez `custom_components/avatar_explorer` dans votre dossier `config/custom_components/`.
2. Copiez `www/avatar-card.js` dans votre dossier `config/www/`.
3. Redémarrez Home Assistant.
4. Ajoutez l'intégration via l'interface.
5. Ajoutez la carte dans votre Dashboard en utilisant le type `custom:avatar-card`.
